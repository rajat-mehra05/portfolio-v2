## Overview

YouTube Transcript API is a lightweight JavaScript/TypeScript library for extracting transcripts and subtitles from YouTube videos without requiring API keys or browser automation tools like Selenium.

## Tech Stack

- **TypeScript** - Type-safe development
- **Axios** - HTTP requests
- **Commander.js** - CLI framework
- **http-proxy-agent** - Proxy functionality
- **Jest** - Testing framework

## Key Features

### Core Functionality
- **Multiple Caption Types**: Supports both manual and auto-generated captions
- **Language Flexibility**: Retrieve transcripts in various languages with priority ordering
- **Translation Support**: Translate transcripts to different languages when available
- **No Authentication**: Works without YouTube API keys

### Output Formats
- **JSON** - Structured data with timestamps
- **Plain Text** - Simple text extraction
- **SRT** - SubRip subtitle format
- **WebVTT** - Web Video Text Tracks format

### Advanced Features
- **CLI Tool**: Command-line interface for quick transcript extraction
- **Proxy Support**: Generic proxy and Webshare rotating proxy integration
- **Error Handling**: Comprehensive handling for video unavailability, rate limiting, and network issues

## Installation

```bash
npm install youtube-transcript-api-js
```
